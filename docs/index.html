ARGUS

Claude Code, Copilot, Github on your desk.

A 480×480 ESP32-S3 dashboard that shows your Claude Code usage in real time. Plug it in, flash it, point the daemon at it.

Flash your device

Connect via USB-C, then click below. Your browser does the rest.

Waveshare ESP32-S3-Touch-LCD-4B

Flash from UART. If flashing fails: hold BOOT, tap RESET, release BOOT, then click Flash again to force download mode.

Web Serial is only supported by Chromium-based browsers (Chrome, Edge, Opera) on desktop.

Download the daemon

A small background process that reads your Claude Code credentials, polls Anthropic for rate-limit headers, and pushes live usage to the device over BLE (or USB-C — pass --serial to auto-detect the board).

Windows argus-daemon.exe macOS argus-daemon Linux argus-daemon

macOS / Linux: run chmod +x argus-daemon first. On macOS, Gatekeeper may require right-click → Open the first time (these builds are unsigned).
